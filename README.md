- MyBatisPlus
    - `@TableName` / `@TableId`
    - `IService<T>` / `ServiceImpl<M, T>` / `Page<T>`
    - `baseMapper`
    - `selectOne()` / `selectPage()` / `selectList()`
    - `saveOrUpdate()` / `removeByIds()`
- Swagger
    - `@Api`
    - `@ApiOperation`
    - `@ApiParam`

- 文件上传
    - `@RequestPart`
    - `MultipartFile` / `transferTo()`
- Lombok
    - `@Data`
    - `@AllArgsConstructor`
    - `@NoArgsConstructor`
- `public` 文件夹
